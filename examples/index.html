Styling

HTML

DOM Data Attributes

Steps live directly beside their targets with data-swz-tour, data-swz-title, group, order, and margin attributes. 

<button
  data-swz-tour="Save the profile"
  data-swz-title="Save"
  data-swz-order="3">
  Save
</button>
View attribute example
JS

JS Object Array

Steps are declared in JavaScript, so targets can be selectors, DOM elements, or omitted entirely for a centered dialog. 

new swz.SenangWebsZero({
  steps: [
    { title: 'Intro', content: 'Welcome', target: '#hero' }
  ]
});
